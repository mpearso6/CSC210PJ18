### **How to run:**
- There are too folders of note
  - The Client folder
  - The Server folder
- Download **nodejs**

#### Client:
1. `cd` into the client folder
2. `npm start` or `npm nodemon` for hot reloading

#### Server:
1. `cd` into the server folder
2. `npm run start`
